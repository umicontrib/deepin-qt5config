Deepin Qt5 Config
===============

Deepin-qt5config is a configuration preferences use to configure Qt feature on deepin platform.

Installation
------------

```bash
# Assuming needed dependencies have been satisfied
# * qt5-qmake
# * qt5-default
#-------------------

mkdir build
cd build
qmake ..
make
sudo make install
```
